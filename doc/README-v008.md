  # 08 - 기본 문법의 활용

1) 사용자의 명령에 따라 회원 등록 및 목록 출력을 처리하도록 변경하라.

- App.java (App.java.01)
    - 사용자로부터 명령을 입력 받는 부분을 추가한다.
    - 조건문을 사용하여 명령어 따라 처리를 분기하도록 변경한다.
    - `quit` 명령이 입력 될 때 실행을 종료하게 한다.
    - 일치하는 명령이 없을 경우 안내 메시지를 출력하게 한다.

실행 결과

```
명령>

명령> /client/add
ID :
PWD : 
이름 : 
생일 :
성별 : 
전화번호 : 
거주지 : 
가입일 :

저장하였습니다.

명령> /client/list
D, 이름, 생일, 성별, 전화번호, 가입일

명령> /client/view
실행할 수 없는 명령입니다.

명령> quit
안녕!
```
