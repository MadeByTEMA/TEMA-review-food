# 04 - 리터럴과 콘솔 출력 다루기

1) 한 명의 회원 정보를 출력한다.
client.java 실행 결과

```
아이디 :
페스워드 :
이름 :
생일 : 
성별 : 
전화번호 : 
주소 : 
가입일 : 
```

아이디, 이름, 생일, 성별, 전화번호, 가입일 출력


2) 한 개의 서울 음식정보를 출력한다.
writingReview.java 실행 결과

```
분류 : 
가게명 : 
메뉴명 : 
가격 : 
맛 별점 :  
양 별점 :
후기 : 
```
가게명, 메뉴, 가격, 맛 별점, 양 별점,

3) 한 개의 서울 지역구를 출력하라.

foodReview.java 실행 결과

```
가게명 : 
메뉴명 :
가격 : 
별점총합 : // 추후 추가
맛 별점 :  
양 별점 :
가격 별점 : // 추후 추가
```
